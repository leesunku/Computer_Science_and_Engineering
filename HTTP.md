# HTTP(HyperText Transfer Protocol)

### HTTP(초본문전송규약, 하이퍼본문전송규약)는 WWW 상에서 정보를 주고받을 수 있는 프로토콜이다.
### 주로 HTML 문서를 주고 받는데 쓰인다. TCP와 UDP를 사용하여, 80번 포트를 사용한다.
### HTTP는 클라이언트와 서버 사이에 이루어지는 요청/응답(request/reponse) 프로토콜이다.
### 클라이언트인 웹 브라우저가 HTTP를 통하여 서버로부터 웹페이지나 그림 정보를 요청하면, 서버는 이 요청에 응답하여,
### 필요한 정보를 해당 사용자에게 전달하게 된다.
